## How to use:

### AutoWheelSpins

- Launch a first spin
- Launch the script
- Set focus on Forza

### AutoGPSDestination

- SetUp full assist
- Set your destination
- Launch the script
- Set focus on Forza

### AutoLabReplay

Will redo the last lab race done.
Can be started from menu or race.

- SetUp full assist
- Launch the script
- Set focus on Forza

### AutoCarBuy

- Place on the car you want to buy in car collection
- Launch the script
- Set focus on Forza

## Requirements

- Forza Horizon 5
- Fullscreen
- Res: 2560x1440
- Language: French
