# Py-ForzaHorizon5-Tools 

## How to use:

### AutoWheelSpins

- Launch a first spin
- Launch the script
- Set focus on Forza

### AutoGPSDestination

- SetUp full assist
- Set your destination
- Launch the script
- Set focus on Forza

## Requirements

- Forza Horizon 5
- Fullscreen
- Res: 2560x1440
- Language: French
